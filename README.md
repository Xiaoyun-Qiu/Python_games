1.如何进行游戏：
    打开游戏界面后，可以直接进行游戏。
    若判断方格无雷，鼠标单击该方格即可，方格会显示以此格为中心的9个个内有几个雷。如果判断错误，点开了有雷的方格，则玩家游戏失败，弹出题为“Bomb！！！”的窗口。玩家点击“Quit”按钮，退出游戏；点击“New Game”按钮，返回游戏界面，开始新游戏。
    若判断方格有雷，要先点击“Setbomb”按钮，再点击该方格，则方格上会标注有雷。如果判断正确，则游戏界面右上方的“mines”处会显示雷数减少：如果判断错误，显示的雷数不变，此时可通过按“Undo”按钮来撤销操作，则最近一次标注的雷的图标会消失，可以继续游戏。
    当所有有雷的方格都被正确标注后，玩家胜利，会弹出一个标题为“Congratul-ations!”的窗口，询问玩家是否继续进行游戏。点击“Quit”按钮，退出游戏，点击“New Game”，返回游戏界面，继续新游戏。
    游戏过程中，如果玩家想开始新游戏，可通过点击“Start”按钮，进行新游戏。
    任何时候，如果玩家不想继续进行游戏，只需点击“Quit”按钮，即可退出游戏。


    
2.游戏设置：
    玩家可通过设置来调整游戏难度。游戏自带三种难度可供玩家选择，分别是easy,middle,hard。除此之外，玩家还可以自行设置游戏难度——设置游戏的行、列以及雷数。
    如果玩家要设置游戏难度，要先点击“New Game”按钮，点击以后，可分别通过点击“Easy”“Middle”“Hard”“Set”按钮来设置简单、中等、难、以及自定义的难度。
